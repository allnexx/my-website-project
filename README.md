1. НАЗВАНИЕ ПРОЕКТА
Барбершоп «GoodBoy»
2. ДЛЯ КОГО ЭТОТ САЙТ
Сайт предназначен для мужчин и подростков, которые хотят быстро выбрать мастера, ознакомиться с работами и записаться на стрижку онлайн без звонков и ожидания.
3. КАКИЕ СТРАНИЦЫ БУДУТ НА САЙТЕ 
    1. Главная – страница содержит основную информацию об организации (контактные данные, адрес) и кнопку «Записаться», которая переносит на следующую страницу;
    1. Каталог услуг – на этой странице клиент может выбрать услуги из предложенного перечня, которые может исполнить далее выбранный им мастер;
    1. Выбор мастера – предлагается список работающих специалистов для подбора наиболее подходящего для каждого клиента, так же присутствует кнопка «Далее», при нажатии на которую открывается следующая страница;
    2. Выбор даты, времени для записи и форма для заполнения личных и контактных данных клиента;
    3. О нас – страница хранит портфолио мастеров, а также данные о заведении (дату основания, оценку и отзывы клиентов) 
4. ЧТО УМЕЕТ ДЕЛАТЬ САЙТ
Сайт способен хранить работы специалистов (в качестве портфолио), обрабатывать заявки для записи, отображать информацию об актуальных услугах и датах приема.

5. ТЕХНОЛОГИИ
LAMP — это набор программ: Linux (операционная система), Apache (веб-сервер), MySQL (база данных), PHP (язык для сайтов).
WordPress — это система для создания сайтов. 
GitHub — это платформа для хранения кода и файлов проекта.
