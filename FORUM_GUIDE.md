# Руководство по установке форума (bbPress)
**Сайт:** Барбершоп «GoodBoy»
**Дата:** 06.05.2026

## 1. Установка
Плагины → Добавить новый → поиск «bbPress» → Установить → Активировать

## 2. Создание форума
Название: Общий форум GoodBoy
Описание: Обсуждение стрижек, ухода за бородой, вопросы мастерам

## 3. Темы (3 шт.)
1. Как часто нужно стричься?
2. Какой воск для укладки выбрать?
3. Стрижка под ноль — актуально?

## 4. Сообщения (3 шт.)
- В теме №1: «Раз в 3-4 недели»
- В теме №2: «Uppercut — лучший»
- В теме №3: «Классика, добавь fade»

## 5. Тестирование
| Тест | Результат |
|------|-----------|
| Форум открывается | ✅ |
| Темы создаются | ✅ |
| Сообщения отправляются | ✅ |
